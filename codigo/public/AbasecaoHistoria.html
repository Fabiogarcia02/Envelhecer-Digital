Dúvidas Frequentes

O que é Wi-fi?
Wi-Fi é como uma "ponte invisível" que liga o seu celular, computador, tablet ou outros dispositivos à internet. Ele funciona através de um roteador, que recebe o sinal da internet por cabo e o transforma em sinal sem fio, permitindo que os aparelhos próximos se conectem.
Como fazer uma senha segura de aplicativo?
Uma senha segura é essencial para proteger os dados dos usuários em qualquer aplicativo. Ela dificulta o acesso não autorizado e evita invasões. Para ser considerada segura, a senha deve atender aos seguintes requisitos: 1. Ter pelo menos 8 a 12 caracteres; 2. Conter letras maiúsculas e minúsculas; 3. Incluir números; 4. Ter caracteres especiais como !, @, #, %, entre outros; 5. Não usar informações óbvias como nomes, datas de nascimento ou sequências simples (ex: "123456" ou "senha"). Além disso, é recomendado que os usuários não reutilizem a mesma senha em vários serviços e, sempre que possível, utilizem autenticação em dois fatores (2FA). Para aplicativos que armazenam senhas, é fundamental que os desenvolvedores não guardem a senha em texto puro, e sim utilizem técnicas de criptografia, como o uso de algoritmos de hash (bcrypt, por exemplo). Adotar essas práticas aumenta muito a segurança dos dados e diminui os riscos de ataques.
Como faz para acessar o banco pela internet?
Acessar um banco de dados pela internet significa conectar-se a um local onde as informações estão armazenadas, mas que não está no seu computador, e sim em outro lugar na rede (ou em servidores online). Existem algumas formas de fazer isso: 1. Utilizando um serviço de hospedagem na nuvem: Onde o banco de dados já está preparado para acesso remoto com segurança (exemplos: Amazon RDS, Google Cloud SQL, MongoDB Atlas). 2. Liberando o acesso remoto de um servidor próprio: Se o banco estiver instalado em um computador pessoal ou servidor local, é preciso abrir as portas corretas do roteador, configurar o banco para aceitar conexões externas e garantir a segurança. 3. Criando uma API intermediária: A forma mais segura é criar um sistema (API) que receba as solicitações dos usuários pela internet e, internamente, faça a consulta ao banco de dados. Assim, o banco nunca fica exposto diretamente. Em todos os casos, é fundamental garantir a segurança com o uso de senhas fortes, criptografia e controle de acesso.
O que é um aplicativo?
Um aplicativo é um programa que você instala ou acessa para realizar uma tarefa no computador, celular, tablet ou web.
